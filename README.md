# kojima-webapp

児島競艇場の予測 Web アプリ（Next.js + Vercel）。

データは姉妹リポジトリ `kojima-boatrace` から CSV 経由で取得し、
3連単の買い目候補ランキング・条件別フィルター・的中率/回収率の
パターン探索を提供する。

## ステータス
スケルトン作成中。次ステップで Next.js プロジェクトを `npx create-next-app` 相当で生成し、
大村アプリ（omura-webapp）と同じ構成で実装予定。

## 設計メモ
- データソース: GitHub raw / Vercel KV / Vercel Edge Config いずれか（運用後決定）
- 主要画面:
  1. 当日のレース別 3連単買い目候補（確率付きランキング）
  2. 条件別フィルター（級別・コース・風・潮位・モーター2連率 等）
  3. 過去パターン探索（年度別・条件別の的中率/回収率）
